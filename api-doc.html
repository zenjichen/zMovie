Tài liệu API OPhim

Chào mừng bạn đến với tài liệu API của OPhim. Đây là hệ thống API RESTful cung cấp dữ liệu phim ảnh phong phú và được cập nhật liên tục.

Base URL:

https://ophim1.com

Định dạng dữ liệu:

JSON

Mã hóa:

UTF-8

Phương thức HTTP:

GET
